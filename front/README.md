# Rouba Montes | frontend

- Criado com CRA (create-react-app);

## Para rodar:
- Certifique-se de ter o <i>NodeJS</i> instalado em sua maquina;

- Navegue até o diretório front: 

  <code>cd front</code>

- Inicialize o projeto instalando as dependências:

  <code>npm install</code>

- Rode a aplicanção com:

  <code>npm run start</code>
